GIT adalah salah satu version control system yang dapat mencatat setiap perubahan file atau sourcode yang diciptakan oleh Linus Torvalds.

Manfaat dari GIT yaitu bisa menyimpan seluruh versi source code, bisa memudahkan membangun sistem bersama programmer-programmer lain, bisa berkolaborasi pada suatu proyek, dan dapat mentracking perubahan source code.
