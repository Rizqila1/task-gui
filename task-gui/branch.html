Brach pada dasarnya adalah cabang/copyan dari repository, manfaatnya sebagai parallel development dan digunakan untuk manajemen pengembangan. Biasanya digunakan saat mengembangkan fitur baru. Perintah dasar : git branch = melihat semua branch local git branch --all = melihat semua branch global/public git branch (nama) = untuk membuat branch baru git checkout (nama branch lain) = untuk berpindah ke branch yang lain git branch -d (nama) = untuk menghapus branch git merge (nama branch lain) = untuk menggabungkan commit dari 2 branch dengan syarat semua file harus sudah dicommit ada juga conflict yaitu ketika terjadi perubahan file yang sama oleh 2 branch yang berbeda, conflict harus diresolve dahulu agar bisa di commit kembali.
