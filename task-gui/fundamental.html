Berisikan perintah dasar yaitu :
git init = untuk menginisialisasi repository, ini sebagai langkah awal wajib untuk menggunakan git.
git config = untuk mengkonfigurasi email dan username (bisa digunakan secara local maupun global).
git config --local (untuk local source) user.name rizqila(nama username)
git config --global (untuk global source) user.email ikilatr@gmail.com(email anda).
git status = untuk mengetahui status file yang secara umum ada 3 tahapan yaitu Untracked-staging area-modified
git add .= untuk memasukan file ke staging area.
git commit -m "judul commit"= untuk mencatat perubahan yang dilakukan.
git log = untuk melihat riwayat commit.
git clone (url) = untuk mencopy repository dari server (github/gitlab/dll).
git push origin (nama branch)= untuk mengirim commit ke server.
git remote -v = semacam api untuk berinteraksi dari client dengan server.
git origin (nama branch) = untuk mengambil commit dari server jika ada perubahan yang dibuat oleh branch lain.
